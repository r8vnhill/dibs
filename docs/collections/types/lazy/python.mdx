En **Python**, los **generadores** ofrecen una manera elegante y eficiente de construir iteradores que producen valores bajo demanda. 1. **Creaci√≥n Simple**: Los generadores se definen como funciones normales que contienen al menos una instrucci√≥n `yield`.

## üåÄ Ejemplo de un Generador en Python

<Explanation>
    La primera l√≠nea crea una lista entera con los 50 valores generados, lo que puede consumir mucha memoria si el rango es muy grande. En cambio, la segunda l√≠nea **no crea la lista completa**, sino que devuelve un generador que calcula cada valor solo cuando se necesita. Esto permite **evaluaci√≥n perezosa** con una sintaxis compacta.
</Explanation>

Puedes convertir ese generador en una lista cuando lo necesites:


Esto es √∫til cuando primero quieres trabajar con un flujo perezoso, pero eventualmente necesitas materializar los datos.
