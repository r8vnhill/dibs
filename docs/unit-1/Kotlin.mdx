# Kotlin

**Kotlin** es un lenguaje de programación multiplataforma desarrollado por **JetBrains** que combina características de la programación orientada a objetos y funcional. Se destaca por su sintaxis concisa y su capacidad para compilar en múltiples plataformas:

- **JavaScript (JS)** y **WebAssembly (WASM)** para la ejecución en navegadores,
- **Java Virtual Machine (JVM)** para servidores y aplicaciones Android,
- **LLVM** para aplicaciones de escritorio y sistemas embebidos.

En este curso, nos centraremos en **Kotlin for the JVM**, que es la plataforma más utilizada para este lenguaje. Sin embargo, los conceptos y técnicas que aprenderás son aplicables a otras plataformas que soporta Kotlin y también te serán útiles en otros lenguajes de programación modernos.

## A Taste of Kotlin

A continuación, te presentamos un ejemplo sencillo en Kotlin para que puedas familiarizarte con su sintaxis y estilo:

```kotlin
data class Person(        // (1)
    val name: String,
    val age: Int? = null  // (2)
)

fun main() {
    val persons = listOf( // (3)
        Person("Harrier Du Bois"),
        Person("Kim Kitsuragi", age = 43) // (4)
    )
    val youngest = persons.minByOrNull { it.age ?: Int.MAX_VALUE }  // (5)
    println("The youngest is: $youngest") // (6)
}
// Output: The youngest is: Person(name=Kim Kitsuragi, age=43)
```

### Explicación del código:

1. Se declara una **data class** llamada `Person` con dos propiedades: `name` de tipo `String` y `age` de tipo `Int?` (opcional).
2. La propiedad `age` tiene un valor predeterminado de `null`.
3. Se crea una lista inmutable de personas con dos elementos.
4. El segundo elemento de la lista tiene un valor nombrado para `age` de 43.
5. Se utiliza `minByOrNull` junto con el operador de elvis (`?:`) para encontrar la persona más joven de la lista.
6. Se imprime el resultado interpolando la variable `youngest` en una cadena de texto.
