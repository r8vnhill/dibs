4. Martin, Robert C., ed. Clean code: a handbook of agile software craftsmanship. Upper Saddle River, NJ: Prentice Hall, 2009.
5. Michael Pilquist, Rúnar Bjarnason, Paul Chiusano, y Paul Chiusano. Functional programming in Scala. Second edition. Shelter Island: Manning Publications, 2023.
