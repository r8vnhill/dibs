En este curso, utilizaremos varias herramientas que facilitarán el desarrollo y manejo de proyectos. A continuación, te mostramos algunas herramientas **necesarias** y **recomendadas**. No es obligatorio que instales todas, pero te sugerimos que elijas aquellas que mejor se adapten a tu flujo de trabajo. Si bien puedes optar por diferentes opciones, las recomendadas son las más comunes en la industria y te permitirán seguir el curso sin problemas.

## Herramientas Necesarias:

1. **[Git](https://git-scm.com/):** Sistema de control de versiones distribuido. Es fundamental para gestionar versiones de tu código y para publicar paquetes en GitHub o colaborar con otrxs desarrolladorxs.
2. **[Java Development Kit (JDK)](https://www.java.com/es/) (versión 8 o superior):** Necesario para compilar y ejecutar programas en Kotlin.
3. **[Kotlin](https://kotlinlang.org/):** El lenguaje principal que utilizaremos en el curso.
4. **[Gradle](https://gradle.org/):** Herramienta de automatización de compilación y gestión de dependencias. Te ayudará a compilar y ejecutar programas en Kotlin de manera eficiente.

## Guía de Instalación

Te proporcionamos aquí una guía para instalar las herramientas necesarias en tu sistema operativo, así como algunas sugerencias adicionales para personalizar tu entorno de desarrollo. Si alguna de las instrucciones no funciona o tienes problemas, consulta la documentación oficial de la herramienta o abre un *issue* en el [repositorio del curso](https://github.com/r8vnhill/dibs) para que podamos ayudarte.

### Windows

#### Windows Terminal

Para mejorar tu experiencia en la terminal de Windows, te recomendamos instalar **Windows Terminal**:

1. Abre la [Microsoft Store](https://www.microsoft.com/es-mx/p/windows-terminal/9n0dx20hk701) y busca "Windows Terminal".
2. Haz clic en "Obtener" para instalar la aplicación.

Si deseas personalizar tu terminal, puedes usar [Oh My Posh](https://ohmyposh.dev/).

#### Scoop

Instala **Scoop**, un gestor de paquetes que te facilitará la instalación de herramientas:

1. Abre _Terminal_ (no como administrador) y ejecuta el siguiente comando:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
# Prueba la instalación
scoop help
```

#### Git

Instala **Git** con **Scoop** ejecutando:

```powershell
scoop install git
# Prueba la instalación
git --version
```

#### GitHub CLI

Para trabajar de manera fluida con GitHub desde la terminal, instala **GitHub CLI** con **Scoop** ejecutando:

```powershell
scoop install gh
# Prueba la instalación
gh --version
```

#### Java Development Kit (JDK)

Instala el **JDK** con **Scoop** ejecutando:

```powershell
scoop bucket add java
scoop install openjdk21
# Prueba la instalación
java -version
```

#### Gradle

Instala **Gradle** con **Scoop** ejecutando:

```powershell
scoop install gradle
# Prueba la instalación
gradle -v
```

#### Kotlin (IntelliJ IDEA)

Para programar en **Kotlin**, instala **JetBrains Toolbox** con **Scoop** ejecutando:

```powershell
scoop install jetbrains-toolbox
```

Luego, abre **JetBrains Toolbox** y busca **IntelliJ IDEA Community** o **IntelliJ IDEA Ultimate**, según prefieras. **Kotlin** está incluido en ambas versiones.

### macOS

#### Homebrew

**Homebrew** es un gestor de paquetes en macOS que facilita la instalación de herramientas. Instálalo ejecutando:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# Prueba la instalación
brew --version
```

#### Git

Instala **Git** con **Homebrew** ejecutando:

```bash
brew install git
# Prueba la instalación
git --version
```

#### GitHub CLI

Para gestionar tus repositorios de GitHub desde la terminal, instala **GitHub CLI** ejecutando:

```bash
brew install gh
# Prueba la instalación
gh --version
```

#### Java Development Kit (JDK)

Instala el **JDK** con **Homebrew** ejecutando:

```bash
brew install temurin@21
# Prueba la instalación
java -version
```

#### Gradle

Instala **Gradle** con **Homebrew** ejecutando:

```bash
brew install gradle
# Prueba la instalación
gradle -v
```

#### Kotlin (IntelliJ IDEA)

Instala **JetBrains Toolbox** con **Homebrew** ejecutando:

```bash
brew install --cask jetbrains-toolbox
```

Luego, abre **JetBrains Toolbox** y selecciona **IntelliJ IDEA Community** o **Ultimate** para comenzar a programar en **Kotlin**.

### Linux

En Linux, la instalación varía dependiendo de tu distribución, pero aquí te mostramos ejemplos comunes para Debian y Ubuntu.

#### Git

Instala **Git** con el gestor de paquetes de tu distribución. En _Debian_ y _Ubuntu_ puedes ejecutar:

```bash
sudo apt install git
# Prueba la instalación
git --version
```

#### GitHub CLI

Para instalar **GitHub CLI** en Debian o Ubuntu, ejecuta el siguiente script:

```bash
(type -p wget >/dev/null || (sudo apt update && sudo apt-get install wget -y)) \
	&& sudo mkdir -p -m 755 /etc/apt/keyrings \
	&& wget -qO- https://cli.github.com/packages/githubcli-archive-keyring.gpg | sudo tee /etc/apt/keyrings/githubcli-archive-keyring.gpg > /dev/null \
	&& sudo chmod go+r /etc/apt/keyrings/githubcli-archive-keyring.gpg \
	&& echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null \
	&& sudo apt update \
	&& sudo apt install gh -y
# Prueba la instalación
gh --version
```

#### Zip y Unzip

Instala **Zip** y **Unzip** con el gestor de paquetes de tu distribución. En _Debian_ y _Ubuntu_ puedes ejecutar:

```bash
sudo apt install zip unzip
```

#### SDKMAN!

**SDKMAN!** es una herramienta útil para gestionar versiones de software como Java y Gradle. Instálala ejecutando:

```bash
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
# Prueba la instalación
sdk version
```

#### Java Development Kit (JDK)

Instala el **JDK** con **SDKMAN!** ejecutando:

```bash
sdk install java
# Prueba la instalación
java -version
```

#### Gradle

Instala **Gradle** con **SDKMAN!** ejecutando:

```bash
sdk install gradle
# Prueba la instalación
gradle -v
```

#### Kotlin (IntelliJ IDEA)

Descarga **JetBrains Toolbox** desde el [sitio web oficial](https://www.jetbrains.com/toolbox-app/download/), y selecciona **IntelliJ IDEA Community** o **Ultimate** para comenzar a trabajar en **Kotlin**.
